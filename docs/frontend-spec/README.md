# AgentOps 前端规范体系 v1

## 一、规范目标

本规范用于统一 AgentOps 多端前端设计、交互、工程与落地标准，支撑平台持续扩展。

## 二、规范目录

### 00-overview
前端规范总纲、设计原则、产品形态分类

### 01-foundation
设计 token、颜色、字体、间距、状态、反馈、可访问性、文案规范

### 02-engineering
前端架构、目录结构、配置中心、组件开发、页面开发、API/状态、Codex 协作、验收规范

### 03-web-admin
后台管理平台规范，包括布局、导航、Dashboard、表格、表单、详情页、Trace/观测页、权限与密度规范

### 04-mobile-admin
移动端管理规范，包括卡片流、轻操作、告警与任务处理

### 05-ai-chat-client
AI 对话客户端规范，包括会话布局、消息类型、输入交互、输出渲染、历史会话与工具调用可观测

### 06-gen-creative-platform
生成式创作平台规范，包括工作台布局、生成流程、参数面板、结果画廊、任务队列与资产管理

## 三、当前优先级

当前优先落地：

1. 00-overview
2. 01-foundation
3. 02-engineering
4. 03-web-admin

## 四、一句话总结

> AgentOps 前端规范体系 v1 的核心目标是：把界面、交互、工程和 AI 协作统一成一套可扩展、可维护的系统。
