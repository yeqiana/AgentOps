# 布局与导航规范

## 一、标准页面结构

```text
AppLayout
├── Sidebar
├── Header
├── PageNavigation
├── PageHeader
├── PageHint（可选）
├── PageContent
└── Footer
```

## 二、Sidebar 规范
- 支持默认展开/收起
- 支持记忆状态
- 当前菜单高亮
- 层级清晰

## 三、Header 规范
- 模块标题
- 模块简述
- 全局搜索
- 环境标签
- 用户菜单

## 四、PageNavigation 规范
- Header 下方独立存在
- 支持面包屑
- 复杂页支持返回入口

## 五、Footer 规范
- 必须统一存在
- 信息弱化但位置稳定
