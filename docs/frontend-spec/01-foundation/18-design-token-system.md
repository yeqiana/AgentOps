# 18-Design Token System（设计令牌体系）

---

## 一、目标

定义 AgentOps 前端统一的视觉基础变量（Design Tokens），用于：

- 保证视觉一致性
- 支撑密度与层级规范（39）
- 降低样式分散与重复定义
- 让 AI 能稳定生成符合规范的 UI

---

## 二、设计原则

### 1. Token 优先于样式

组件不得直接写死：

- 颜色
- 间距
- 字号
- 圆角

必须通过 Token 使用。

---

### 2. Token 必须表达语义，而不是数值

❌ 不推荐：

```css
--gray-100
--blue-200
