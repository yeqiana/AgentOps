# FAQ

## 1. 为什么上传文件后没有触发 OCR？

可能原因：
- `run_tools=false`
- `OCR_TOOL_PATH` 未配置
- 本机未安装 `tesseract`

## 2. 为什么视频没有做 ASR？

可能原因：
- `VIDEO_AUDIO_TOOL_PATH` 未配置
- `ASR_TOOL_PATH` 未配置
- 本机未安装 `ffmpeg` 或 `whisper`

## 3. 为什么默认上传目录是 `/app/download`？

原因：
- 这是统一默认值，便于容器化和企业部署口径一致
- 同时支持通过 `APP_DOWNLOAD_DIR` 覆盖

## 4. 为什么当前数据库还是 SQLite？

原因：
- 阶段 1 目标是先形成闭环
- 后续可切 PostgreSQL

## 5. 如何查失败任务？

查询：
- `GET /tasks?status=failed`
- `GET /tasks/{task_id}`

## 6. 现在是否已经支持请求路由中台？

答：
- 还没有完整支持
- 当前仅支持基于输入资产类型的最小工具路由
- 独立请求路由中台已进入正式设计和开发计划

## 7. 现在是否已经支持异步任务体系？

答：
- 还没有完整支持
- 当前执行链仍以同步执行为主
- 当前已具备 `task_id` 和任务查询基础
- 异步任务体系已纳入后续阶段规划

## 8. 现在是否已经支持权限控制？

答：
- 还没有完整支持
- 当前版本未实现统一鉴权、资源级授权和工具级授权
- 权限控制已纳入正式设计与开发计划

## 9. 现在是否已经支持多 Agent 编排？

答：
- 还没有完整支持
- 当前仅支持单 Agent 主流程
- 多 Agent 编排已纳入正式设计与开发计划

## 10. 为什么文档里写了这些能力，但系统里还没有？

原因：
- 企业级正式文档同时覆盖当前能力、设计预留和阶段计划
- 文档中已经区分“当前已支持”和“后续建设”
- 评审与对外说明时应以各文档中的状态说明为准
