# 用户手册

## 1. 适用对象

面向：
- 业务使用者
- 项目接入方
- 测试人员

## 2. 可以做什么

当前系统可用于：
- 文本对话
- 图片分析
- 音频分析
- 视频分析
- 文档分析
- 任务与资产追踪

## 3. 使用入口

### CLI

启动：

```powershell
.\.venv\Scripts\python.exe run.py
```

### API

启动：

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.presentation.api:create_app --factory --host 127.0.0.1 --port 8011
```

## 4. 常见操作

### 文本对话

输入普通文本即可。

### 上传文件

使用 `POST /assets/upload`。

### 查询任务

使用：
- `GET /tasks`
- `GET /tasks/{task_id}`

### 查询资产

使用：
- `GET /sessions/{session_id}/assets`
- `GET /assets/{asset_id}`

## 5. 注意事项

- 上传目录默认是 `/app/download`
- 视频工具依赖本地 `ffmpeg / ffprobe`
- OCR 依赖本地 `tesseract`
- ASR 依赖本地 `whisper`
