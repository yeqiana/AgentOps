# 功能清单

## 1. 已交付功能

### 1.1 基础对话与输入能力

- CLI 连续对话
- CLI 流式回答
- HTTP 对话接口
- SSE 流式对话接口
- 文本、图片、音频、视频、文件输入
- 上传型资产入口
- 资产分析入口

### 1.2 工具与多模态能力

- 工具注册中心
- Agent 自动调用工具
- OCR
- ASR
- 视频探测
- 视频抽帧
- 视频抽音轨
- PDF 解析

### 1.3 任务与查询能力

- 同步任务执行
- 异步任务提交与事件回查
- 任务状态查询
- 任务事件查询
- 异步运行时快照查询
- 会话查询
- 资产查询

### 1.4 路由与编排能力

- 请求路由中台最小版
- 路由配置快照查询
- 路由决策查询
- 路由统计查询
- 最小多 Agent 编排
- 正式角色协议
- 可切换执行协议

### 1.5 治理与安全能力

- 统一鉴权
- 最小 RBAC
- 限流
- 幂等
- 工具白名单
- 上传大小限制
- 上传类型限制

### 1.6 可观测与恢复能力

- trace service
- 告警记录
- 告警查询
- trace 关联告警查询
- 模型重试
- 工具重试
- 模型熔断
- 工具熔断
- 模型降级到 `mock`
- 工具 `soft-fail`

### 1.7 配置中心能力

- 运行时配置查询
- 运行时配置写入
- 数据库覆盖环境变量
- 数据库驱动角色注册

## 2. 当前预留能力

- 异步任务后台执行器
- 异步任务状态链
- 异步任务事件持久化
- 异步任务运行时快照
- 队列平台演进位

## 3. 未完成能力

- 完整 RBAC
- 完整异步任务平台
- 可视化 trace
- 观测面板
- 成本与配额治理
