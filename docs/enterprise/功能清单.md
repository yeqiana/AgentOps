# 功能清单

## 1. 文档说明

本文档用于汇总 AgentOps 当前版本已经落地的能力、部分落地的能力和后续预留能力。

当前口径：
- 阶段 1 已完成
- 阶段 2 开发中，当前进度约 `93%`

## 2. 已落地能力

### 2.1 对话与接入能力

- CLI 持续对话
- HTTP API 对话
- SSE 流式对话
- 普通回答输出
- 流式回答输出

### 2.2 多模型与多模态能力

- OpenAI 兼容协议接入
- `mock` 模式
- 图片输入
- 音频输入
- 视频输入
- 文件输入
- 上传文件标准化
- PDF 正文提取

### 2.3 工具能力

- 工具注册中心
- Agent 自动调用工具
- OCR 工具接入
- ASR 工具接入
- 视频探测工具接入
- 视频抽帧工具接入
- 视频抽音轨工具接入
- 工具白名单控制

### 2.4 治理能力

- 统一鉴权
- 限流
- 幂等
- trace service
- 告警记录
- 告警查询
- trace 与告警关联查询
- trace 聚合摘要查询

### 2.5 恢复能力

- 模型重试
- 工具重试
- 模型熔断
- 工具熔断
- 模型降级到 `mock`
- 工具 `soft-fail`
- 配置化恢复策略

### 2.6 多 Agent 编排能力

- `router`
- `debate`
- `arbitration`
- `critic`
- `review`
- 正式角色协议
- 可切换执行协议

### 2.7 配置与管理能力

- 数据库驱动运行时配置中心
- 数据库驱动角色注册
- 工作流配置查询
- 运行时配置查询与更新
- 恢复配置查询
- 安全配置查询

### 2.8 数据与排障能力

- 会话落库
- 消息落库
- 资产落库
- 任务落库
- 工具结果落库
- trace 落库
- alert 落库
- 任务查询
- 异步任务取消
- 任务事件查询
- 会话查询
- 资产查询
- 工具结果查询

## 3. 部分落地能力

- 请求路由中台
  - 当前已有 workflow 级路由
  - 尚未形成独立路由中台

- 权限控制
  - 当前已有统一鉴权
  - 尚未形成完整 RBAC

- 异步任务体系
  - 当前已有任务模型和任务查询
  - 尚未形成队列、worker 和异步提交体系

- 策略中心
  - 当前已有运行时配置中心和角色配置
  - 尚未形成完整策略中心与变更审计体系

## 4. 预留能力

- 可视化 trace
- 观测与监控面板
- 成本与配额治理
- 模型路由与策略引擎
- 完整异步任务平台
- 完整权限治理体系

## 5. 当前结论

当前版本已经不是单一聊天示例工程，而是一个具备治理能力、恢复能力、排障能力和最小多 Agent 编排能力的 Agent 底座。
