# 功能清单

- 阶段 2：已完成并结项

## 2. 阶段 1 已交付能力

- CLI 与 HTTP API 双入口
- 会话、消息、资产、任务、工具结果持久化
- 多模态输入解析与上传
- PDF 解析
- OCR、ASR、视频探测、抽帧、抽音轨
- 工具网关与自动工具节点
- 统一执行上下文标识
- 结构化错误响应
- 自动化测试基线

## 3. 阶段 2 已交付能力

### 3.1 治理能力

- 统一鉴权：`API Key`、`Bearer Token`
- 最小 RBAC
- 主体授权关系回查
- 角色权限矩阵查询
- 限流与幂等
- LLM 与工具的重试、熔断、降级
- 恢复告警记录、查询与统计

### 3.2 trace 与排障能力

- trace 落库与查询
- trace 统计聚合：`GET /traces/stats`
- trace 聚合摘要：`GET /traces/{trace_id}/summary`
- trace 时间线：`GET /traces/{trace_id}/timeline`
- trace 图谱：`GET /traces/{trace_id}/graph`
- trace 与 alert 关联查询：`GET /traces/{trace_id}/alerts`
- 控制台聚合总览：`GET /operations/overview`

### 3.3 编排能力

- 请求路由中台最小版：`request_route_service`
- 路由配置、预览、查询与统计
- 最小多 Agent 编排：
  - `router`
  - `debate`
  - `arbitration`
  - `critic`
  - `review`
- 正式角色协议：
  - `support`
  - `challenge`
  - `planner`
  - `executor`
  - `arbitration`
  - `critic`
  - `reviewer`
- 可切换执行协议：
  - `delegated`
  - `standard`

### 3.4 配置中心能力

- 数据库驱动运行时配置中心：`sys_runtime_config`
- 数据库覆盖环境变量
- 配置变更审计与回查：`GET /config/runtime/events`
- 安全快照：`GET /security/config`
- 恢复快照：`GET /recovery/config`
- 路由快照：`GET /routing/config`
- 工作流快照：`GET /workflow/config`

### 3.5 流式与异步任务能力

- CLI 流式输出
- SSE 流式对话：`POST /chat/stream`
- 异步任务提交：`POST /tasks/submit`
- 异步任务运行时快照：`GET /tasks/runtime`
- 异步任务取消：`POST /tasks/{task_id}/cancel`
- 异步任务重试：`POST /tasks/{task_id}/retry`
- 任务事件持久化：`biz_task_event`
- 任务事件查询：`GET /tasks/{task_id}/events`

### 3.6 聚合查询能力

- 任务聚合摘要：`GET /tasks/{task_id}/summary`
- 会话聚合摘要：`GET /sessions/{session_id}/summary`
- 任务状态统计：`GET /tasks/stats`
- 路由统计：`GET /routes/stats`
- 告警统计：`GET /alerts/stats`
- trace 统计：`GET /traces/stats`
