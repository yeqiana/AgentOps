# 运维手册

## 1. 适用范围

适用于本项目阶段 1 的本地、测试和单机部署环境。

## 2. 环境要求

- Python 3.12
- SQLite
- 可选本地工具：
  - Tesseract
  - Whisper
  - FFmpeg
  - FFprobe

## 3. 启动方式

### CLI

```powershell
.\.venv\Scripts\python.exe run.py
```

### API

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.presentation.api:create_app --factory --host 127.0.0.1 --port 8011
```

## 4. 关键环境变量

- `LLM_PROVIDER`
- `APP_DATABASE_URL`
- `APP_DOWNLOAD_DIR`
- `LOG_ENABLED`
- `LOG_LEVEL`
- `LOG_FILE_PATH`
- `OCR_TOOL_PATH`
- `ASR_TOOL_PATH`
- `VIDEO_PROBE_TOOL_PATH`
- `VIDEO_FRAME_TOOL_PATH`
- `VIDEO_AUDIO_TOOL_PATH`

## 5. 常见运维动作

- 检查 `/health`
- 检查日志
- 检查任务表
- 检查工具是否注册

## 6. 备份建议

- 备份 SQLite 数据库文件
- 备份上传目录
- 备份日志文件
