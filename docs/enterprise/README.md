## 4. 当前口径

当前统一口径如下：
- 阶段 1：已完成并结项
- 阶段 2：开发中，当前进度约 `80% - 85%`
- 阶段 3：规划中

当前已经落地的重点包括：
- 统一鉴权
- 限流与幂等
- trace service
- 恢复告警与 trace 关联查询
- 重试、熔断、降级与恢复策略
- 数据库驱动运行时配置中心
- 数据库驱动角色注册
- 最小多 Agent 编排
- 正式角色协议
- 可切换执行协议
- CLI 与 API 流式对话
