- 阶段 1：已完成并结项

阶段 2 当前已落地重点：
- 请求路由中台最小版
- 统一鉴权与最小 RBAC
- 限流与幂等
- trace 与告警治理
- 重试、熔断、降级、恢复策略

- 最小多 Agent 编排
- 正式角色协议
- 可切换执行协议
- CLI 与 SSE 流式对话
- 异步任务提交预留与后台执行器
