# 培训材料

## 1. 培训目标

让开发、测试和接入方理解：
- 项目是什么
- 怎么跑
- 怎么查
- 怎么扩展

## 2. 培训对象

- 开发
- 测试
- 接口接入方
- 运维

## 3. 培训提纲

### 第一部分：项目定位

- Agent 底座是什么
- 阶段 1 做到了什么

### 第二部分：系统结构

- 五层结构
- tool_node / plan_node / answer_node

### 第三部分：使用方式

- CLI
- API
- 上传
- 查询

### 第四部分：排障方式

- trace_id
- task_id
- tool_results

### 第五部分：后续扩展

- router
- review
- multi-agent
