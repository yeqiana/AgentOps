# 操作指南

## 1. 快速启动

### 启动 CLI

```powershell
.\.venv\Scripts\python.exe run.py
```

### 启动 API

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.presentation.api:create_app --factory --host 127.0.0.1 --port 8011
```

## 2. 快速测试

### 健康检查

```powershell
Invoke-RestMethod http://127.0.0.1:8011/health
```

### 聊天

```powershell
$body = @{
  message = "帮我写一句简短的自我介绍"
  user_name = "api-user"
  session_title = "API Session"
} | ConvertTo-Json
Invoke-RestMethod http://127.0.0.1:8011/chat -Method Post -ContentType "application/json" -Body $body
```

### 上传

```powershell
curl -X POST "http://127.0.0.1:8011/assets/upload" `
  -F "file=@D:\demo\image.png" `
  -F "kind=image" `
  -F "prompt=请提取图片里的关键信息" `
  -F "run_tools=true"
```

### 测试全部用例

```powershell
.\.venv\Scripts\python.exe -m unittest discover -s tests -p "test_*.py" -v
```
