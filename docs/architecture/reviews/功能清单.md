# 功能清单

## 文档目的

这份文档用于汇总阶段 1 已交付能力，方便评审、汇报和后续规划。

## 一、运行入口

### CLI

- 持续对话 REPL
- 支持多轮输入
- 支持退出命令

### HTTP API

- `GET /health`
- `POST /chat`
- `GET /sessions`
- `GET /sessions/{session_id}`
- `GET /sessions/{session_id}/assets`
- `GET /sessions/{session_id}/tasks`
- `GET /assets/{asset_id}`
- `GET /tasks`
- `GET /tasks/{task_id}`
- `GET /tools`
- `POST /tools/{tool_name}/execute`
- `POST /assets/analyze`
- `POST /assets/upload`

## 二、模型能力

- 支持 OpenAI 兼容协议
- 支持多 Provider 扩展结构
- 支持 `mock` 模式
- 保留 LLM 分包结构

## 三、多模态输入能力

### 命令式输入

- 图片
  - `/image-file`
  - `/image-url`
  - `/image-object`
  - `/image-base64`

- 音频
  - `/audio-file`
  - `/audio-url`
  - `/audio-object`
  - `/audio-base64`

- 视频
  - `/video-file`
  - `/video-url`
  - `/video-object`
  - `/video-base64`

- 文件
  - `/file-path`
  - `/file-url`
  - `/file-object`
  - `/file-base64`

### 上传型输入

- `POST /assets/upload`
- `multipart/form-data`
- 默认上传目录 `/app/download`
- 支持环境变量 `APP_DOWNLOAD_DIR`

## 四、媒体解析能力

### 图片

- 本地路径读取
- URL 读取
- Base64 读取
- 对象存储 URI 标准化
- 标准化为图片资产

### 音频

- 本地路径读取
- URL 读取
- Base64 读取
- WAV 元数据解析
- 标准化为音频资产

### 视频

- 本地路径读取
- URL 读取
- Base64 读取
- MP4/MOV/AVI 基础容器解析
- 标准化为视频资产

### 文件

- 文本文件解析
- DOCX 正文提取
- PDF 正文提取
- 标准化为文件资产

## 五、工具网关能力

### 已支持工具

- `python_echo`
- `ocr_tesseract`
- `asr_whisper`
- `video_ffprobe`
- `video_ffmpeg_frame`
- `video_ffmpeg_audio`

### 已支持行为

- 工具注册
- 工具发现
- 工具直调
- 统一 stdout / stderr / exit_code 结构
- 失败结果标准化

## 六、工作流能力

当前主链路：

```text
tool_node -> plan_node -> answer_node
```

### 已支持自动工具后处理

- 图片进入 OCR
- 音频进入 ASR
- 视频进入 ffprobe
- 视频进入关键帧抽取
- 关键帧回灌为图片资产
- 关键帧继续 OCR
- 视频抽音轨
- 抽取音轨继续 ASR
