# API 文档索引

## 说明

本目录用于存放对外接口协议、调用约束和后续 API 演进资料。

当前阶段的 API 主入口见：
- [操作指南](../enterprise/操作指南.md)
- [用户手册](../enterprise/用户手册.md)
- [功能清单](../enterprise/功能清单.md)

当前已落地接口包括：
- `GET /health`
- `POST /chat`
- `GET /sessions`
- `GET /sessions/{session_id}`
- `GET /sessions/{session_id}/assets`
- `GET /sessions/{session_id}/tasks`
- `GET /assets/{asset_id}`
- `GET /tasks`
- `GET /tasks/{task_id}`
- `GET /tools`
- `POST /tools/{tool_name}/execute`
- `POST /assets/analyze`
- `POST /assets/upload`
